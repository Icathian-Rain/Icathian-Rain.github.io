1. git log

1.1 description

show the commit logs

功能：

  • Commit Limiting：过滤提交
  • History Simplification : 简化提交历史的
  • Commit Ordering: 修改默认提交排序的
  • Commit Formatting : 修改默认提交显示格式的
  • Diff Formatting: 控制提交显示文件的差异格式的
